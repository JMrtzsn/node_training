Description:
Node.js Express App interacting with Datastore.
Running a Cloud build and deploy to App Engine on any push to Source Repository.
Datastore contains "Customer" which has entities with 5 custom properties.

Usage:
Get all customers:
https://molnkul-269517.appspot.com/getCustomers

Get customer by ID:
https://molnkul-269517.appspot.com/getCustomers/["ID"]
ex: https://molnkul-269517.appspot.com/getCustomers/4793933427113984
